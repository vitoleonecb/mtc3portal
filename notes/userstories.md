--------------------------- --------------------------- Roles

The Activity Meter is measuring the user's activity dynamically over time. The Meter decides the level of users role.
The idea is to allow the users to get a lay of the land before gaining extra influence on the production and workshops. If they walk away for a while they will have to earn the permissions back by engaging.

--------------------------- User

--- Use Email and Password to Login
--- Use Account Settings to Change Notification Settings
--- Use Account Settings to Manage Anonymous Response Sharing Setting (Admin Only, Plus Analysts, Plus New Users)
--- Use Workshops to View All Workshops and Their Statuses (Attending, Scheduled, Completed)
--- Use Modules to View All Modules and Their Statuses (Completed, Open)
--- Use a Module to Respond to Prompts (if open) or View Theirs' and others' Past Responses (if completed)
--- Use Profile to check their activity meter. This metric will decide the level of privileges they have.

--------------------------- Analyst + User

--- Use Prompts to view all responses to prompts
--- Use Prompts to make comments on responses

--------------------------- Writer + Analyst + User

--- Use Modules to view all of the responses and comments
--- Use Modules to Make New Prompt proposals

--------------------------- Administrator + Writer + Analyst + User

--- Use Workshops to Create Workshops
--- Use Modules to Create Modules and Approve Prompt Proposals
--- Use Prompts to Create Prompts and Review Analyst Comments
--- Use Analytics to View the Attendance for Workshops

There are three groups of users. The user who can respond to prompts and set their notification settings. 
Each step up in the activity monitor results in greater influence on the final production decisions made.
That way the level of power is delegated based on merit.

User Responds to prompts.
Analyst Comments on prompts.
Writer Synthesizes prompt responses and their comments to Write a Prompt Proposal.
Admin Approves the Prompt Proposal and thus becomes a prompt.
Every user has a Profile page that can be used to check their activity meter. This metric will decide the level of privileges they have.
